# Welcome To Quiz App!
This repo is my personal SPA-project, using Vue.js.
You can check your computer science knowledge doing this app:)


## Project setup
Install npm.
```
npm install
```
Clone this repo.

```
git clone https://github.com/mox692/Vue_Quiz_App.git
```

Run server
```
npm run serve
```
